🎵 Melodify Music Player

Melodify is a simple Spotify-style music player built using React.
It allows users to play songs, search music, and navigate through a sidebar interface.

---

🚀 Features

- ▶️ Play / Pause songs
- ⏭ Next / Previous controls
- 🔍 Song search functionality
- 🎧 Playlist section (Coming soon)
- 📱 Responsive UI
- 🎨 Spotify-style layout

---

🛠 Tech Stack

- React.js
- JavaScript
- HTML / CSS
